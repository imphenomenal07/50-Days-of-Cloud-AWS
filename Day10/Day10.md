# Day 10: Attach Elastic IP to EC2 Instance

# Step1: Login into AWS Console with the given credentials

# Step2: Open EC2 dashboard

AWS Console > Search 'EC2' > Open EC2 Dashbaord > Elastic IPs > Select the required 'Elastic IP' > Actions > Assocaite Elastic IP address > Select required 'EC2 Instance' > Allocate/save

# Step3: Go back to running instances and refresh, Elastic IP will be available for the running the instance!!
